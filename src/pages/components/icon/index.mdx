## Props

| Prop Name       | Type                    | Default     | Required? | Description |
| --------------- | ----------------------- | ----------- | --------- | ----------- |
| `icon`          |	`keyof typeof IconSVGs` | `undefined` |	Yes	      | Determines icon displayed, see possible values above |
| `className`	    | `string`                | `undefined` |	No	      | Adds a class to icon |
| `ariaHidden`    |	`boolean`               | `False`	    | No        | Determines if icon is hidden to screenreader |
| `ariaLabel`	    | `string`                | `undefined`	| No        | Label to be read by screenreader |
| `ariaLabelledby`|	`string`                | `undefined`	| No        | DOM ID of object that labels icon |
